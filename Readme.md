# Code Setup

1. Initialize the project

   ```bash
   pnpm init
   ```

2. Initialize TypeScript configuration

   ```bash
   tsc --init
   ```

3. Generate `.gitignore`

   ```bash
   pnpm dlx gitignore Node
   ```

   Or

   ```bash
   npx gitignore Node
   ```

4. Install tsc-watch

   ```bash
   pnpm add -D tsc-watch
   ```

# Express setup

1. pnpm i express

2. pnpm i @types/express

3. pnpm i @types/node
